# ❤️ Testando um pouco de front-end (e sobrevivendo)

Esse projeto é basicamente o resultado de:
👉 curiosidade + vontade de aprender + um pouquinho de sofrimento 😅

A ideia foi criar um coração animado usando várias frases **"eu amo você"**, só pra testar conceitos básicos de front-end na prática.

---

## 🤔 O que isso faz?

* Gera várias frases "eu amo você" automaticamente
* Usa matemática pra formar um coração
* Aplica animações pra deixar tudo mais bonito e menos parado
* Faz o coração “bater” igual um coração de verdade 💓

---

## 🧠 O que eu pratiquei aqui

* Estrutura com HTML
* Estilização e animações com CSS
* Lógica com JavaScript
* E um pouco de paciência (parte mais difícil)

---

## 🚀 Tecnologias usadas

* HTML
* CSS
* JavaScript

---

## ▶️ Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador
3. E pronto: um coração aparece na tela ❤️

## 🔄 Atualizações

### 💓 Versão 2.0 — Coração com batimento

Depois da versão inicial, o projeto recebeu algumas melhorias:

* Adicionei uma animação de **batimento (pulse)** no coração
* Ajustei o tempo das animações para que tudo ficasse mais natural
* Sincronizei o movimento das palavras com o batimento
* Resultado: o coração agora parece mais “vivo” e menos robótico 😄

---

### 🧪 Versão 1.0 — Versão inicial

* Criação do coração com frases "eu amo você"
* Posicionamento usando equação matemática
* Animação simples de flutuação

---

## 💭 O que mudou na prática?

Antes:
👉 As palavras se moviam de forma independente

Agora:
👉 Tudo se move de forma sincronizada, criando um efeito mais realista de batimento 💓
